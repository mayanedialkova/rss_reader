# README

## A very basic app for consuming RSS feeds

* Ruby version: ruby 2.5.0p0
* Rails version: Rails 5.1.5
* Database: sqlite3

## Steps needed to run the application:
```
git clone git@github.com:mayanedialkova/rss_reader.git
cd rss_reader
bundle install
rake db:create
rake db:migrate
rails s
```
